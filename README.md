# Minimal API CRUD em C#

API simples criada com .NET usando Minimal API.

Permite realizar operações CRUD:
- Criar
- Listar
- Atualizar
- Deletar

Os dados são armazenados em SQLite.

## Tecnologias

- C#
- .NET
- Minimal API
- SQLite
- Entity Framework Core

## Como executar

Clone o repositório:

git clone https://github.com/seu-usuario/seu-projeto

Entre na pasta:

cd seu-projeto

Execute:

dotnet run
